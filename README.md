# TMDB Movie Browser

A small educational React project for browsing movies using the TMDB API.

## Features

- Movie list with pagination
- Movie details page
- Search by movie title
- Filter by genres
- Light/dark theme switcher
- Responsive movie cards

## Tech Stack

- React
- TypeScript
- React Router
- Axios
- CSS Modules
- Bootstrap
- MUI Rating
